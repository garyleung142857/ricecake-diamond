#### 2!n Game Force

Responder has a hand that does not want to sell out at 3N if there is a fit, or a hand that has an alternative game other than 3N if 3N is flawed.

- 5M with short suit / weak doubleton

#### Confit

Responder has a balanced hand, investigates the number of controls and a fit for a slam.

<BiddingTree>
1!n - 2!s; 2!n/3!c - 3!d
- 3!h: 2-3 controls
    - 3!s: Ask again
        - 3!n: 2 controls
        - 4!c: 3 controls, 4+ suits. Bid up the line
    - 3!n: To play
    - 4!c+: 4+ suits. Bid up the line
- 3!s: 4 controls
- 3!n: 5 controls
- 4!c+: 6+ controls. 4+ suit. Bid up the line
</BiddingTree>

<Aside type="note">
The number of control is based on 10-13 range. For 14-16 1!n, add one control to the requirements.
</Aside>

Then the development follows the following rules:
- Any time the responder rebids NT, the opener must pass
- The opener must not raise the level unless fitting
- Both players bids suits up the line, rebidding shows 5
- Fit partner by a raise
